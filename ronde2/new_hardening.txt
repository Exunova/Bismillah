1. password and login policies
-password
bisa edit /etc/security/pwquality.conf
vim /etc/security/pwquality.conf
terus tinggal ganti parameter
minlen = 8:
Panjang karakter
minclass = 1:
kelas karakter
maxrepeat = 0:
Batasan pengulangan karakter
maxclassrepeat = 0:
Batasan pengulangan kelas karakter
lcredit = 0:
jumlah lowercase
ucredit = 0:
jumlah uppercase
dcredit = 0:
jumlah digit
ocredit = 0:
jumlah non alfa-numerik

bisa setup satu satu

setup length
# authconfig --passminlen=8 --update
check
# grep "^minlen" /etc/security/pwquality.conf

setup lowercase
# authconfig --enablereqlower --update
check
# grep "^lcredit" /etc/security/pwquality.conf
kalua lcredit = -1 berarti wajib ada lowercase

setup uppercase
# authconfig --enablerequpper --update
check
# grep "^ucredit" /etc/security/pwquality.conf

setup digit
# authconfig --enablereqdigit --update
check
# grep "^dcredit" /etc/security/pwquality.conf

setup non-alpha-numerik character
# authconfig --enablereqother --update
check
# grep "^ocredit" /etc/security/pwquality.conf

Deny reused passwords: remember=X, where X is the number of past passwords to be
denied. For example:
password sufficient pam_unix.so sha512 shadow nullok try_first_pass
use_authtok remember=5

password expiration date
edit di /etc/login.defs
vim /etc/login.defs
PASS_MAX_DAYS: The maximum number of days a password can be used
PASS_MIN_DAYS: The minimum number of days allowed between password changes
PASS_MIN_LEN: The minimum acceptable password length
PASS_WARN_AGE: The number of daysâ€™ warning to be given before a password expires

edit in /etc/pam.d/system-auth for specific usecase

5. Firewall policies
cek
sudo firewall-cmd --state
sudo firewall-cmd --list-all
firewall-cmd --get-services

add service
sudo firewall-cmd --zone=public --add-service=http
cek 
sudo firewall-cmd --zone=public --list-services
buat permanen menggunakan --permanent
sudo firewall-cmd --zone=public --permanent --add-service=http
cek semua permanen service
sudo firewall-cmd --zone=public --permanent --list-services

buka port
sudo firewall-cmd --zone=public --add-port=5000/tcp
cek
sudo firewall-cmd --zone=public --list-ports
adding in range
sudo firewall-cmd --zone=public --add-port=4990-4999/udp
pake --permanent untuk buka permanen

diy service sendiri
buat file
sudo vim /etc/firewalld/services/example.xml
contoh
<?xml version="1.0" encoding="utf-8"?>
<service>
  <short>Example Service</short>
  <description>This is just an example service.  It probably shouldn't be used on a real system.</description>
  <port protocol="tcp" port="7777"/>
  <port protocol="udp" port="8888"/>
</service>
reload 
sudo firewall-cmd --reload
cek
firewall-cmd --get-services
